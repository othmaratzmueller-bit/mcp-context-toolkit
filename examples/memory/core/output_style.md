# Output style & interaction

## prefer_terse_replies
*Keep answers short and scannable — bullets/tables over prose.*

The user reads fast and dislikes long prose. Default to tight, structured
answers. Expand only when explicitly asked. Related: [[no_filler_apologies]].

## no_filler_apologies
*No "I'm sorry / as an AI" filler — just correct and continue.*

When wrong, fix it and move on without a paragraph of apology. State what
changed and why in one line.
